## 一、都市脑洞

### 范例：《病娇妈妈们找上门，儿女都吓哭了》
**开头结构**：场景→独处→意外→反转→系统

> 江城市。破旧楼房内。苏牧下班后没回家。他跟老婆江亦瑶撒了个谎，说要出差。实际上，他躲到了这里。

**结构拆解**：
1. 2字交代地点（江城市）
2. 5字交代场景（破旧楼房内）
3. 一句话交代状态（下班没回家）
4. 一句话制造悬念（撒谎，躲）
5. 然后展开：意外女人出现→仙人跳→离婚→系统激活

**可套用模板**：
```
{城市名}。
{简短场景描述}。
{主角}{一个反常行为}。
{一句话解释为什么这么做，但留悬念}。
实际上，{真相，制造更大悬念}。
```

## 二、豪门总裁

### 范例A：《禁她入骨》
**开头结构**：极端环境→感官描写→身份交代→被卖真相

> 东南亚，帕孔地区。盛夏。烈日高悬，炙烤大地。空气又干又烫，每一口呼吸都如在吞咽含沙的热风。四周散发着尿液的骚臭味，汗液的酸腐味。

**结构拆解**：
1. 地点+季节（东南亚，盛夏）
2. 温度+嗅觉感官
3. 困境（铁笼，被当猪仔）
4. 主角身份（大二学生，被骗）
5. 背叛者揭晓（亲叔叔）

**可套用模板**：
```
{极端/异国地点}。{季节}。
{2-3个感官细节}。
{困境描写，1-2句}。
{主角身份交代}。
{一句话揭示谁害了她}。
```

### 范例B：《不愿再让你低头》
**开头结构**：医院→诊断结果→倒计时→电话

> 出了医院，浓重的夜色压下来，秋风卷着落叶吹来，刚从医院出来的岳绮颜下意识的拢了拢风衣，握着诊断书的手指因为用力骨节泛出青白色。

**结构拆解**：
1. 医院出口（暗示坏消息）
2. 天气描写映射心情
3. 握诊断书的动作（不说内容，制造悬念）
4. 医生话引用（最多两年半）
5. 打电话给丈夫（核心关系）

**可套用模板**：
```
出了{医院/法院/某场所}，{天气映射心情}。
{主角}{一个微动作暗示内心}。
脑海中闪过{某人}的话——
"{冲击性的一句话}"
{主角的反应}。
{一个动作推动下一步}。
```

## 三、古代宅斗

### 范例：《十年前他们要我顶替妹妹进青楼》
**开头结构**：核心事件→被弃→回归→反差

> 债主把我和妹妹卖去西域青楼抵债。爹娘凑到的钱只够赎一人，他们毫不犹豫的救了妹妹。

**结构拆解**：
1. 一句话核心设定（被卖）
2. 父母选择（救妹妹不救我）
3. 十年后对比（妹妹入东宫 vs 我成摄政王）
4. 回归场景（走正门）
5. 父母反应（嫌弃→恐惧）

**可套用模板**：
```
{N年前，被害/被弃的事件}。
{亲人选择救别人不救主角}。
"{被弃时的伤人台词}"
{N年后，主角已{隐藏身份/实力}}。
{回归场景，制造身份反差}。
```

## 四、玄幻修仙

### 范例A：《西游你叫我吗喽》
**开头结构**：世界危机→日常切入→反差主角→系统/穿越

> "听说了吗，樱花国京都，被一只青毛巨狮占据……"

**结构拆解**：
1. 群众对话交代世界观
2. 危机背景（西游妖魔入侵现实）
3. 学校场景（日常反差）
4. 主角报名（所有人觉得他疯了）
5. 穿越者优势（知道西游记剧情）

**可套用模板**：
```
"{一条震惊的新闻/传闻}"
"{群众反应/讨论}"
——安静——
{场景转换：学校/公司/日常}
{权威人物宣布一个危险任务}
{所有人退缩}
"我报名！"
{主角站出来，所有人震惊}
{暗示主角有隐藏优势}
```

### 范例B：《长生火影》（动漫衍生）
**开头结构**：原作角色互动→穿越者内心→系统选择

**可套用模板**：
```
"{原作角色名}的声音{描述}"
{原作角色向主角提出请求/挑战}
主角内心：{穿越者视角的分析}
系统/面板出现：
【选择一：{选项}】奖励：{xxx} 代价：{xxx}
【选择二：{选项}】奖励：{xxx} 代价：{xxx}
主角选择+理由。
```

## 五、青春校园

### 范例：《你携盛夏，渡我寒冬》
**开头结构**：苦难叠加→遇见→改变

> 最走投无路那年，我遇到了林燃。我被同学欺负，被亲妈揍，被邻居们骂野种。

**结构拆解**：
1. 倒叙钩子（最走投无路那年）
2. 三重苦难叠加（欺负+家暴+歧视）
3. 遇见（他/她出现）
4. 第一印象（不良少年形象）
5. 交易（我帮你写作业，你保护我）

**可套用模板**：
```
最{负面状态}那年，我遇到了{对方}。
我{被欺负}，被{家人伤害}，被{社会歧视}。
{对方}则{不良/叛逆行为列举}。
我走投无路。
"{交易/请求台词}"
"{对方的反应，出人意料}"
```

## 六、现实情感

### 范例：《五年恩情，一朝两清》
**开头结构**：纪念日被放鸽子→揭穿→回忆契约

> 顾泽言的白月光林薇回国那天，是我们结婚三周年纪念日。我坐在餐厅靠窗的位置，看着桌上冷透的牛排和燃尽的蜡烛。

**结构拆解**：
1. 两个事件叠加（纪念日 vs 白月光回国）
2. 场景细节（冷牛排、燃尽蜡烛=被遗忘）
3. 手机消息（"临时有急事"）
4. 朋友圈验证（白月光发图）
5. 契约婚姻背景交代

**可套用模板**：
```
{对方}{重要事件}那天，是{我们的重要日子}。
{被忽视的场景细节，暗示等待}。
手机屏幕亮了："{敷衍的消息}"
{主角查看{朋友圈/聊天}——真相}。
{一句话内心独白，冷静但心碎}。
```

## 七、点众轻喜剧

### 范例：《捡回来的小乞丐，竟然想咬我》
**开头结构**：吐槽→捡人→反差→笑料→升级

> 许寂看着雪地里那个浑身是血、眼神凶得像要吃人的小罗莉，叹了口气。"这年头，碰瓷的成本都这么高了吗？"

**结构拆解**：
1. 主角吐槽式反应（不是害怕是吐槽）
2. 建立世界观（修仙+穿越10年+没金手指）
3. 捡到危险生物（小萝莉=鬼帝重生）
4. 认知错位制造笑点（鬼帝咬不动凡人）
5. 一碗粥引发升级（种田天赋+灵气）

**可套用模板**：
```
{主角}看着{危险/奇怪的事物}，{非正常反应}。
"{吐槽台词}"
{世界观一句话交代：穿越N年/系统/金手指状态}
{捡到/遇到一个人/物}
{对方的真实身份，与表面反差极大}
{对方的攻击完全无效→制造笑点}
{主角毫无危机感地照顾对方}
{意外发现对方获得好处→双赢设定}
```

## 开头通用规则

### 基本要求
- 第一个一级结构中，主要角色（主角+可攻略对象+主要配角+主要反派）不得登场3个以上
- 开头必须以叙事为主，压缩介绍性、抒情性和环境描写文字
- 叙事优先于塑造人设，人设尽量通过叙事手法体现
- 叙事连贯：正叙为主，不用插叙，不切换视角，不用回忆/梦境/大篇幅对话
- 绝不水字数——对话是水字数重灾区，开头两章不写介绍信息的对话，只写推进剧情的对话
- 第一章不上来就写打斗（小说需要大量信息铺垫才能在打斗中产生爽感）
- 300字内主角必须登场，1000字内必须出现爽点或期待点
- 不写序章/楔子/引（除非叙事为主、足够简短且与第一章有密切逻辑联系）
- 第一章由主角行动引导故事走向，主角不能"打酱油"
- 世界观详细解说至少等到第一个一级结构结束后
- 爽文第一章内表明主角有装逼的能力或潜力

### 开局环境要求
- 第一个环境应无需动脑、无需过多设定解释就能理解
- 最好有一个方便多角色出场的环境，便于调动人物、发展剧情

### 开头布局要求
- 氛围应轻松明快或平顺自然，切忌开头悲伤压抑
- 切入点（触发点）应在第一个一级结构中出现，最好在第一章
- 切入点是叙事主角人生的转折点，之后主角进入非常态生活
- 情绪安排：尽快迎来第一次上行。推荐"先短暂下行再上行"或"原地起飞"
- 第一个冲突应影响重大，事先点明解决的好处和不解决的坏处
- 主角行动必须带来正面收益，最好是主动选择使用"神器"解决问题
- 第一章必须说明主角当下的目标（短期目标要紧迫，长期目标要引人期待）
- 第一章必须写明本文的卖点

### 开头期待感三路径
- 事业线突破口：设定/素材新颖，读者预见喜闻乐见结局但猜不到过程
- 爱情线突破口：人设或人设搭配新颖有吸引力，开头就抛出亮点
- 双线缠绕突破口：事业线与爱情线在第一个一级结构同时进行，互相影响

### 开头噱头类型
- **事件噱头**：以一个事件快速进入故事（常见于飞卢）
- **金手指噱头**：先描写主角困境，引出目标，金手指第一次施展
- **人设噱头**：不建议新手使用，出彩人设极少
- 事件噱头和金手指噱头的写法不同，不能混用
- 噱头吸引读者后，立刻嫁接主线（主角的目标），事件结束后必须马上接上新目标

### 噱头三分法详解

**事件噱头写法**：
- 书名即事件，如"账号被找回，我反手充值一百万"
- 事件作为切入点，极快进入故事，通常写五章左右
- 强烈建议用日常常见事件开局（天然有代入感）
- 一上来就进入事件，不要铺垫穿越/绑定金手指
- 错误示范：用金手指写法去写事件（读者冲着事件进来，你磨磨唧唧写主角和困境）

**金手指噱头写法**：
- 金手指分布在全文，前期出现多，后期逐渐减少
- 标准流程：描写主角 → 当前困境 → 引出目标 → 金手指第一次施展 → 取得少许成果
- 金手指可以结合事件写法：开局事件安排在金手指出现前，事件拉成主线，由主线诞生目标，再出现金手指

**人设噱头写法**（仅限高阶）：
- 出彩人设能直接影响剧情构建（如慎重勇者，性格决定阶段剧情发展）
- 杀伐果断/冷酷无情只是当下基本要求，不算出彩人设
- 人设噱头多出现在日常文、恋爱文，由人设引发戏剧性
- 新人不推荐：人设写不对就成了深井冰，读者看不懂

### 七步法第一章实例拆解（《从斩妖除魔开始长生不死》）

标准3000字第一章的字数分配：

| 段落 | 字数 | 内容 |
|------|------|------|
| 世界观+穿越事实+前身经历 | 300字 | 快速交代背景，不啰嗦 |
| 第一个吸睛小剧情 | 500字 | 用事件制造初始吸引 |
| 前置金手指获得安全感 | 400字 | 金手指首次展示 |
| 感受到不安全 | 400字 | 新的威胁出现 |
| 遇见危机 | 500字 | 核心冲突到来 |
| 直接使用金手指 | 400字 | 解决问题，完成第一个爽点 |

### 爽点层次划分与信息差抹平技术

**爽点本质**：主角装逼、主角牛逼、配角认知发生变化。注意力放在配角身上——他们之前怎么样，之后又怎么样。

**爽点层次**（从低到高）：

| 层次 | 类型 | 用途 |
|------|------|------|
| 日常装逼 | 生活中随便装一下 | 调剂用，距下个高潮有点远时打出来 |
| 核心爽点 | 切在主线上的装逼 | 主线矛盾解决过程中的装逼 |
| 偏离爽点 | 与主线无关的装逼 | 无意义，应避免 |

**信息差抹平技术**（连环震惊）：
1. 主角做完一件事 → 当场观众震惊
2. 别的地方观众震惊
3. 专家震惊
4. 配角之间信息差抹平 → 再次震惊
5. 层层递进，一章可容纳多个爽点

### 情绪拉扯=斗地主模型

情绪拉扯的核心：主角与反派双方角力的过程。

**模型**：
- 主角出对三，反派出对四（反派稍强）
- 主角出对A，反派出对2（反派再压）
- 主角直接扔王炸！一锤定音！

**关键规则**：
- 经过拉扯后的爽感远比单纯碾压爽
- 每次小角力都以主角稍占上风告终
- 最后迎来主角大胜利
- 主角可以风轻云淡一指灭杀，也可以歇斯底里，但前者更爽
- 主角心态要有逼格：千亿富翁不会和乞丐争得面红耳赤
- 赢麻了不等于一路碾压，一路碾压没有看头

### 逼格心态写作法

写书时幻想主角是一个大人物，不是小混混，是见过大风大浪的人：
- 反派骂主角，主角微微一笑，毫不放在心上（不毒）
- 反派骂主角，主角气的要死（毒点）
- 升级文也能写逼格：升级是升级，心态是心态，两者独立
- 解决问题的方式决定爽感：风轻云淡一指灭杀 >> 歇斯底里艰难取胜

## 书名命名技法

### 身份法（覆盖市面上约1/3书名）

**第一类：高逼格身份**
- 前半句放绝对至高的身份，后半句写爽点变种
- 公式：`[仙帝/魔神/财阀]爹妈 + [反差行为/目标]`
- 范例："武神爹，财阀妈，我当躺赢狗怎么了？"

**第二类：戏剧身份对照**
- 找一对有差距且有关联的身份，做位置置换
- 公式：`[低位身份]替[高位身份]做[超出预期的事]`
- 步骤：找身份对（替身vs鲜肉）→ 找共同目标（要红）→ 置换结果
- 范例："让你当书童，你替少爷科举中状元？"
- 进阶变种：删除高位身份，只留低位身份+超出预期的结果
  - "让你做武打替身，你怎么成功夫巨星了？"

**第三类：身份擦边型**
- 用身份关系制造暧昧暗示，有风险但吸量极强
- 范例："前女友母亲怀上双胞胎"

### 其他书名方法概览

| 方法 | 核心 | 适用平台 |
|------|------|----------|
| 掉马甲型 | 隐藏身份逐渐暴露 | 已过度使用，慎用 |
| 伪同人类 | 原创中加入同人热词元素（甲斗腰带/八奇技） | 番茄/飞卢 |
| 逼格起名法 | 用意境感强的词汇 | 起点 |
| 反义词法 | 正反反差制造好奇 | 通用 |
| 金手指法 | 直接把金手指写在书名里 | 番茄/飞卢 |
| 主线+爽点法 | 主线剧情核心+一个爽点暗示 | 起点 |
| 乐子文法 | 一句话就让人想笑 | 番茄/飞卢 |

### 平台起名差异

| 平台 | 起名风格 | 关键原则 |
|------|----------|----------|
| 番茄 | 噱头优先，吸量第一 | 书名决定80%流量上限，相当于免费抖加 |
| 起点 | 轻度反差+金手指+主线+爽点 | 过度噱头会榨干前期潜力，内容必须贴合书名 |
| 飞卢 | 极致吸量，全网最会起名 | 可以参考飞卢榜单学习起名技巧 |

### 番茄书测改名策略

百万字书测时的五个书名选取法：
1. 金手指相关的一个
2. 第一个单元剧情相关的一个
3. 主线相关的一个
4. 与原书名差一个字的保底款
5. 一个稍微夸张吸量的款

## 简介/文案分类体系（晋江扫榜实战提炼）

### 五种基本文案类型

| 类型 | 结构 | 效果 | 适用 |
|------|------|------|------|
| 四段式成功型 | 情境→冲突→行动→成功 | 给读者安全感，明确知道主角会赢 | 最通用，适合新手 |
| 三段式行动型 | 情境→冲突→行动（截止到行动开始） | 留悬念，读者想知道结果如何 | 事业线突出的文 |
| 二段式悬疑型 | 情境→一个反转/悬念 | 制造好奇，读者想知道真相 | 悬疑/烧脑/冷门题材 |
| 四段式失败型 | 情境→冲突→行动→失败 | 制造虐感和愤怒 | 虐文/追妻火葬场 |
| 三段式成功型 | 情境→转折→成功 | 简洁明快，给安全感 | 甜文/轻松向 |

### 文案组合策略

- 单一型文案：适合主线清晰的作品
- 叠加型文案：两个或多个基本型组合，覆盖不同读者
  - 成功型+悬疑型 = 既有安全感又有好奇心
  - 行动型+成功型 = 先制造期待再满足
  - 失败型+悬疑型 = 虐后留悬念

### 文案写作要点（从扫榜数据验证）

- 文案中必须体现**女主的主体性**——主体性差的作品收藏涨幅普遍低于预期
- 读者需要**安全感**——至少暗示主角会成功/不会一直被虐
- 文案和书名要**货不对板会劝退**——书名暗示的内容文案里必须有
- 悬疑型文案不宜叠加过多——太"谜语人"读者理解不了且安全感不足
- 冷门历史/题材可能劝退——即使文案写得好，门槛过高也会影响收藏

## 章节名策略

### 章节名的核心功能

- 章节名是对本章内容的**核心凝练**，也是**期待感的一部分**
- 高手会通过章节名让读者产生初步代入感
- 很多读者看完标题才决定追不追

### 两种取章节名的时机

| 方法 | 操作 | 适用 |
|------|------|------|
| 先写后取 | 写完内容，挑章节中最有戏剧性、最有梗的一句话当标题 | 有存稿/不分章时更合适，往往有意外好梗 |
| 先取后写 | 先想好有梗的章节名，围绕它写内容 | 需要精准控制节奏时 |

### 章节名实战原则

1. **不要用干瘪的概括**："开始""觉醒""升级"——毫无期待感
2. **要有戏剧性**：用最有梗的一句话或最冲突的信息
3. **写铺垫章也要把标题写得有噱头**：读者一旦看出是铺垫章就会跳过甚至掉追读
4. **选择章节中最装x的一句话做标题**：让读者觉得"主角又要搞事了"
5. **不要乱取标题**：乱取是大神的专利，新人每章标题都要有梗
6. **避免"分析""局势""博弈"等字眼**：读者一眼看出是铺垫就会跑

### 章节名对比范例

```
差：第一章 开始 / 第二章 觉醒 / 第三章 升级
好：第一章 穿越异界 / 第二章 金手指觉醒 / 第三章 境界提升
更好：从正文中提取最抓人的梗做标题
```

## 高潮逆推法 + AB 粗纲

### 高潮逆推法

从确定的高潮场景反推前面需要铺垫的人物和情节：

1. 确定高潮场景（同人基本固定）
2. 反推需要哪些人物在场
3. 反推需要哪些前置铺垫
4. 按 AB 交替排列

### AB 粗纲

- A = 压情绪/铺垫/伏笔
- B = 抬情绪/擦边/小收获
- 按 ABABAB 排列形成粗纲
- 码字时只关注当前的 AB 写好，不必思考长期剧情

**优点**：提升码字速度，节奏好
**副作用**：如果高潮和 AB 难写，可能中途推翻规划
