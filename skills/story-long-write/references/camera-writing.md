# 镜头式写作法

忘掉全知视角，你是跟在主角身后的摄像机。读者只能看到主角看到的东西。

## 核心原则

1. **只写能看到/听到/闻到/摸到的**：不写「他心里想」「他感到」
2. **对话即剧情**：每句对话都要推进剧情、塑造人物、或传递信息
3. **动作代替描述**：「他很紧张」→「他的手在抖」
4. **手机屏幕优先**：短句、频繁换行、每段不超过 3 句

## 禁用清单

### 禁用视角
- ❌ 全知叙述：「此时的他还不知道，三天后…」
- ❌ 上帝视角：「如果她知道真相…」
- ❌ 心理分析：「他内心深处其实…」

### 禁用词汇
仿佛、好像、犹如、一丝、一抹、深吸一口气、缓缓、不禁、眼中闪过、嘴角勾起、眉头微皱、不容置疑、不易察觉

### 禁用句式
- 「...，带着...」（如：他说，带着一丝无奈）
- 「XX说道」→ 用动作神态替代
- 「他/她知道...」→ 直接展示，不告诉
- 「像XX一样的XX」→ 陈词滥调的比喻

## 替代技巧

| 禁用 | 替代 |
|------|------|
| 他感到紧张 | 他的手在抖 |
| 她很伤心 | 她转过身，肩膀微微颤动 |
| "好的。"他说道 | "好的。"他点了根烟 |
| 他深吸一口气 | 他把烟摁灭在烟灰缸里 |
| 她眼中闪过一丝惊讶 | 她的手停在了半空 |
| 他不禁笑了 | 他的嘴角翘了一下 |

## 震惊九法

写装逼打脸时，震惊反应的9种具体写法，避免千篇一律：

| 类型 | 写法 | 核心要点 |
|------|------|----------|
| 普通震惊 | 事件 + A震惊 | 直接反应，最基础 |
| 多人震惊 | A震惊 + B震惊（B身份/实力高于A） | 层层递进，身份越高越能衬托主角 |
| 解说震惊 | 旁白解说招式 + A震惊 | 先用旁白拔高招式逼格，再写角色反应 |
| 画面震惊 | 画面描写 + A震惊 | 天地变色、异象等大场面，再切入角色反应 |
| 对比震惊 | A震惊 + 与自身对比 | 角色拿自己和主角比，越比越绝望 |
| 问号震惊 | A内心追问探究 + A震惊 | 角色先质疑、先不服，再被现实打脸 |
| 变换地点震惊 | 地点1震惊 → 地点2震惊 → 地点3震惊 | 消息扩散式震惊，范围越大越爽 |
| 身心震惊 | 身体反应 + 心理震慑 | 先写生理反应（冰寒、心跳漏拍），再写心理崩溃 |
| 人设震惊 | A之前不屑/自信 → A被打脸后震惊 | 利用角色前后反差制造戏剧性 |

**使用要点**：
- 震惊的基础是信息差，没有信息差不要硬写震惊
- 一段剧情中至少用2-3种不同类型的震惊，避免单调
- 身份越高的人震惊，爽感越强
- 震惊角色的反应要符合其人设，不同性格的人震惊方式不同
- 限制震惊范围：不要每段都让所有人震惊，留有余地避免"震无可震"

### 震惊反应分阶段写法

震惊不是一步到位，而是按层次递进：

1. 信息释放 → 众人初步反应（疑惑/不屑）
2. 更大信息释放 → 众人反应升级（震惊）
3. 权威人物背书/最高潮 → 众人反应最强烈（震撼/膜拜）
4. 对立方垂头丧气离开（情绪压一压，形成完整弧线）

**关键**：装逼最重要的是反应，反应写得多样关键在人设。不同身份、不同立场的人，面对同一件事的震惊方式必须不同。

### 三翻四震

三次反转 + 四次震惊的情绪升级结构：

```
翻1：主角展示第一个成果 → 震惊1（初步）
翻2：主角展示更深层次的成果 → 震惊2（升级）
翻3：主角的终极底牌揭晓 → 震惊3（高潮）+ 震惊4（余波扩散）
```

核心逻辑：每次反转都要比上一次更强，震惊要层层递进而不能平级重复。适用于大高潮场景。

## 信息差构建法

利用信息不对称制造期待感和戏剧性，三种核心模式：

### 单层信息差
读者知道主角厉害，角色不知道 → 期待角色发现真相时的反应

### 双层信息差
- 读者知道主角真实身份
- A不知道，B也不完全知道
- 读者期待A发现时的反应，也期待B知道A被蒙在鼓里后的反应

### 三层信息差（最强）
- 读者期待B见到主角的反应（第一层）
- 读者期待A知道主角身份时的反应（第二层）
- A与B之间也有信息差，读者期待B知道A让他教训的人是大佬后的反应（第三层）

**操作要点**：
- 信息差越多层，读者越离不开，因为每揭一层就是一个爽点
- 每一层信息差的揭示都必须有合适的角色在场做反应
- 揭示信息差时要有节奏，一层一层来，不要一口气全揭开

## 对话写作规范

### 每句对话必须满足至少一个：
1. 推进剧情（透露新信息、做出决定、发生冲突）
2. 塑造人物（展现性格、背景、价值观）
3. 制造张力（威胁、隐瞒、试探）

### 对话格式
- 用动作打断长对话
- 每次说话不超过 3 句
- 用神态/动作代替「他说」「她说」

### 对话时机
- 长篇幅对话要塞在期待点和爽点之间——读者因为期待爽点会容忍中间的对话
- 对话本身可以强化驱动力、制造信息交流受阻、脱离读者预期
- 无意义的聊天和谈情说爱可以省略脑补
- 把部分信息转化为情节、心理描写、旁白、环境、动作等表达
- 适度将"一问一答"模式转化为"一个人主动说，另一个人给反应"

### 对话情绪五级递增

写矛盾冲突时，对话越不礼貌，情绪越强烈：

1. **客观陈述事实** — 最弱
2. **客观陈述 + 提出建议** — 礼貌但有力
3. **主观指责** — 开始有冲突感
4. **主观指责 + 强制命令** — 强情绪
5. **主观指责 + PUA抬升自己** — 最强

最强话术：打着"为你好"的幌子，句句不离关心，但句句都是嫌弃、指责、厌恶

### 对话拉扯技巧
- 一问一答没有情绪波动；要像打斗场景一样让对话有"出招"感
- 对话要有人物性格、隐藏意图、情绪递进，而非直奔主题
- 中间任何一个片段都能结束对话，字数合适即可

### 好的对话
```
"钱呢？"

王少往椅背一靠，翘起二郎腿。"什么钱？"

"上个月的三十万。"林风往前一步。

王少的手指敲了敲桌面。"我说了吗？我不记得了。"
```

### 差的对话
```
"你还欠我三十万块钱，是上个月借的，答应这个月还的。"林风说道。

"哦，那个钱啊，我最近手头有点紧，能不能再等等？"王少不好意思地说道。

"不行，我已经等了一个月了。"林风坚定地说道。
```

## 三机位法

场景写作的三机位交替法，用于控制叙事节奏和画面感：

| 机位 | 视角 | 内容 | 功能 |
|------|------|------|------|
| 机位1 | 近景 | 主角的动作、表情、心理 | 推进主线，读者代入 |
| 机位2 | 远景 | 配角反应、环境变化、围观者 | 扩展信息面，制造反差 |
| 机位3 | 旁白 | 设定补充、背景交代 | 适时补充必要信息 |

### 机位交替原则
- 核心1和2，机位3穿插使用
- 设定都由冲突引出，无关设定不要乱写
- 校对时检查：机位1多了排2，机位2多了排1，穿插3做画外音补充
- 可以主动增删镜头来控制节奏

## 场景与分镜基础

### 场景划分原则
- 场景划分是分镜的基础——不懂场景直接上分镜会显得凌乱
- 一个场景 = 一个地点 + 一个时间段 + 一组人物
- 场景内必须发生至少一个有意义的事件（推进剧情/深化关系/揭示信息）
- 场景结束要有钩子（新信息/新悬念/新冲突）

### 视角转换控制
- 视角转换不要过于频繁——描写一个完整情节后再进入下一个情节点
- 衔接过渡要自然，用相似物/五感/情绪做转场
- 单场景内尽量保持同一视角

### 画面感训练
- 拉片练习：选一段视频/影视剧，写出分镜头内容
- 三种画面层次交替：环境→人物动作→特写细节
- 写之前先在脑中"放映"场景，想清楚机位和画面再动笔

## 间接反应描写深化

### 核心原则

正面描写只是铺垫，侧面反应才是爽点。不要写场面有多厉害，要写看到场面的人有多震撼。

### 操作方法

主角做完一件震撼的事后——戛然而止，不继续描写场面本身，转而写：

1. **当场观众的反应**：各种不同身份的人如何震惊
2. **别处观众的反应**：消息扩散后的连锁反应
3. **专家的反应**：内行看到手法后的震惊（层次更高）
4. **配角之间信息差抹平后的再震惊**：A告诉B，B的固有印象被颠覆

### 烘托式描写范例

> 不写"这饼多好吃"，而是写品尝者的吃相、哄抢的情形、想推荐给朋友的冲动、回过头来还想吃的神态——以此烘托饼有多好吃。

### 应用要点

- 主角行动写到位就停，转写反应
- 反应要有层次：普通人→内行人→高位者，层层递进
- 不同身份的人反应方式必须不同（符合人设）
- 反应中可以自然带出对比和衬托

## 震惊的信息差分层释放

震惊不是一步到位，而是按信息差分层释放，每层都有不同角色的反应：

### 释放层次

```
第1层：主角做完一件事 → 当场观众震惊
第2层：消息扩散 → 别处观众震惊
第3层：内行看到手法 → 专家震惊
第4层：配角之间信息差抹平 → 再次震惊
```

### 操作范例

1. 主角展示财力，震惊配角A
2. 主角又花钱建学校，震惊配角B
3. A和B遇到一起，B告诉A主角建了学校
4. 配角A震惊——因为在他认知中主角是玩世不恭的人，没想到这么有爱心

**核心**：把注意力放在配角身上——他们之前怎么样，之后怎么样。配角之间关于主角的信息差是否还没抹平。

### 关键规则

- 爽点 = 主角装逼 + 配角认知变化
- 每层震惊要有不同的角度和反应
- 一段剧情中至少用2-3种不同类型的震惊
- 身份越高的人震惊，爽感越强
