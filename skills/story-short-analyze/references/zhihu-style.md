# 知乎盐言故事风格识别

> 完整格式规范见 story-short-write 的 [zhihu-format.md](../../story-short-write/references/zhihu-format.md)。本文聚焦于分析/识别知乎盐言风格的写作特征。

## 知乎盐言的核心风格特征

### 1. 第一人称限定视角
- 全部用「我」叙事
- 不出现全知视角的描述
- 读者只能通过「我」的眼睛看世界

### 2. 对话用方括号
- 使用【】而非引号
- 对话独立成段
- 说话人标签简短（他笑了 / 她没说话）

### 3. 短句为主，口语化
- 一句话一个信息点
- 避免长复句
- 日常对话感强

### 4. 紧凑排版
- 段落开头两个全角空格
- 段落之间无空行
- 章节之间一个空行

## 拆文时的盐言特征检查项

| 检查项 | 说明 |
|--------|------|
| 视角一致性 | 是否全程第一人称，有无视角跳跃 |
| 对话格式 | 是否统一使用【】，有无混用引号 |
| 句式密度 | 短句占比是否超过 70% |
| 口语程度 | 是否有书面腔/翻译腔残留 |
| 情绪节奏 | 是否有明确的情绪曲线（盐言最看重情绪拉扯） |
| 反转位置 | 反转是否出现在 70-85% 位置（盐言标准位置） |
| 章尾钩子 | 每章结尾是否有钩子 |

## 常见盐言类型与情绪模式

| 类型 | 核心情绪 | 典型反转 |
|------|----------|----------|
| 虐恋 | 意难平 | 他其实一直爱她 |
| 重生 | 爽感释放 | 这一世不再犯同样的错 |
| 悬疑 | 细思极恐 | 凶手就在身边 |
| 职场 | 共鸣愤怒 | 反杀成功 |
| 婚姻 | 共鸣+治愈 | 离开之后更好 |
| 治愈 | 温暖 | 被忽视的爱被发现 |

## 盐言 vs 其他平台的风格差异

| 维度 | 知乎盐言 | 番茄短篇 | 小红书故事 |
|------|----------|----------|------------|
| 视角 | 第一人称限定 | 不限 | 不限 |
| 篇幅 | 8000-20000字 | 灵活 | 1000-5000字 |
| 核心卖点 | 情绪拉扯+反转 | 爽感节奏 | 话题性+共鸣 |
| 对话风格 | 【】方括号 | 引号 | 自由 |
| 读者期待 | 被虐到想哭，反转治愈 | 一口气读完很爽 | 有话题想转发 |
| 付费模式 | 会员/单篇付费 | 免费阅读 | 免费引流 |
