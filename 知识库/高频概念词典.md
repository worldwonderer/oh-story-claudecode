# 网络小说高频概念词典

## 核心术语

| 术语 | 定义 |
|------|------|
| 黄金三章 | 长篇网文前三章（约 1-1.5 万字），决定签约和追读 |
| 爽点 | 让读者产生满足感的情节点，网文的核心商品 |
| 钩子 | 吸引读者继续阅读的悬念或冲突设计 |
| 追读率 | 持续追看更新的读者比例，长篇最核心的指标 |
| 完读率 | 从头读到尾的读者比例，短篇最核心的指标 |
| 铺放比 | 爽点铺垫字数与释放字数的比例，通常 3:1 到 5:1 |
| 日更 | 每天更新，长篇网文的基本要求（4000-6000 字/天） |
| 上架 | 长篇从免费章节转入付费章节（通常 80-120 章） |
| 签约 | 与平台签订合约，通常需要 30-50 章的考核期 |
| 金手指 | 主角独有的超能力/系统/优势，推动剧情的核心设定 |
| 套路 | 经市场验证的叙事模式，读者熟悉且有期待 |
| 扫榜 | 系统性浏览平台排行榜分析市场趋势 |
| 精修 | 对已完成章节做二次润色和优化 |
| 开书 | 正式开始连载一本长篇网文 |
| 水字数 | 用无意义的内容凑字数，拉低追读率的致命问题 |
| 毒点 | 让读者产生不适或愤怒的情节，可能导致弃书 |
| 毒发 | 读者因为毒点而弃书 |
| 崩文 | 剧情逻辑崩塌或质量严重下降 |
| 烂尾 | 结局草草收场或质量下降 |
| 粮草 | 值得一看的作品 |
| 仙草 | 极其优秀、值得反复阅读的作品 |
| 干粮 | 能看但不出彩的作品 |
| 毒草 | 有严重问题的作品 |
| 宰相 | 宰人（虐主）+ 宰相（大量杀角色），指作者喜欢虐主 |
| 甜宠 | 全程甜蜜无虐的言情风格 |
| 先婚后爱 | 结婚/契约关系后产生真感情的题材 |
| 破镜重圆 | 分手/离婚后重新在一起的题材 |
| 双洁 | 双方都没有感情经历或身体经历的设定 |
| 骨架 | 大纲的基本结构 |
| 细纲 | 每章的详细大纲 |
| 存稿 | 提前写好但尚未发布的章节 |
| 断更 | 停止更新，对长篇杀伤力极大 |
| 爆更 | 一天更新大量章节（通常 3 章以上） |
| 五更 | 一天更新 5 章 |
| 三江阁 | 起点编辑推荐位，重要新书曝光渠道 |

|------|------|

| 快穿 | 主角穿越多个世界完成任务的类型 |
| 无限流 | 角色在多个副本世界冒险的类型 |
| 追妻火葬场 | 渣男追悔莫及的虐恋题材 |
| 死人文学 | 主角死后他人追悔的虐心题材 |
| 白月光 | 不可替代的初恋/理想对象 |
| 替身文 | 替代他人位置的虐恋设定 |
| 大女主 | 以女性成长为核心的爽文 |
| 诸天流 | 穿越多个影视/小说世界的类型 |
| 同人 | 基于已有作品世界观的二次创作 |
| 穿书 | 穿越到小说世界中成为角色 |
| 年代文 | 以 60-90 年代为背景的故事 |
| 宫斗 | 以宫廷权力斗争为核心的女频题材 |
| 权谋 | 以政治斗争为核心的男频题材 |
| 规则怪谈 | 以诡异规则和生存为核心的新兴题材 |
| 第四天灾 | 玩家/异世界人被当作天灾的新兴题材 |
| 脑洞文 | 以奇思妙想为核心的轻松向作品 |
| 凡人流 | 主角天赋普通靠努力成长的类型 |
| 无敌流 | 主角一开始就无敌的爽文类型 |
| 系统面板 | 以数据化属性显示的系统界面 |
| 种马/后宫 | 男主有多个女性伴侣的设定 |
| 修罗场 | 多个感情对象同时出现的尴尬场面 |
| 龙傲天 | 实力逆天、一路碾压的男主 |

| 侧面烘托 | 不直接描写而通过他人反应展现 |
| 白描 | 不加修饰的简洁描写手法 |
